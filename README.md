# Amazon Reviews Dashboard
GUI built in python for exploring [Amazon Review 2018 dataset](https://jmcauley.ucsd.edu/data/amazon/).  
  
# Installation
  

## Dependencies
json  
PySimpleGUI  
numpy  
matplotlib  
tqdm  
```  
$ pip install -r requirements.txt  
```
